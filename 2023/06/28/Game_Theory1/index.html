Game theory

在市场经济条件下，众多自利个体的同时行动如何产生经济结果？

策略性的相互依赖（strategic interdependence）

多人互动的核心特征之一。每个参与人的收益不仅取决于自己的行为，还取决于其他参与人的行为。并且参与人的最优行动可能取决于对其他参与人行动的预期。

分析该局面的工具就是非合作博弈（noncooperative game theory）。

不同的多人互动问题，参与人之间的策略性依赖程度不一样：

  • 垄断情形/完全竞争情形：策略性依赖程度太小。
  • 寡头市场等：策略性依赖程度很高。

博弈的理念

一群人在策略性相互依赖环境下的相互作用的正式描述。博弈的过程需要清楚四件事情：

  1. 参与人（players）：谁参与了博弈？
  2. 规则（rules）：谁在何时行动？他们在行动时知道什么信息？他们能做什么？
  3. 结果（outcomes）：对于参与人们每组可能的行动组合，相应的结果是什么？
  4. 收益（payoffs）：参与人们在各个可能结果上的偏好（即效用函数）是什么样的？

有些博弈中，一方所得就是另一方所失，这就是零和博弈（zero-sum games）。更多的博弈形式是不限于完全或部分冲突的情形。

博弈的展开形表示法

已知上述四种元素，便可以用展开形（extensive form）表示博弈。展开形依赖于博弈树（game tree）
